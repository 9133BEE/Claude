## 策略備註

### 王牌組合
- **再行銷 CPL NT$560、80 潛客**：中彰投再行銷是整個帳戶表現最強的組，建議持續放大，補充新素材維持頻率
- **投資/半退休/休閒 CPL NT$297**：最低 CPL，北屯十期受眾對「投資+半退休生活」訴求高度共鳴
- **科技自營商族群**：中科水湳中清地區科技自營商（test版）14潛客、CPL 407，為精準核心受眾

### 有效受眾方向（CPL < NT$1,000）
1. **再行銷**：王牌，CPL 560，80 潛客
2. **投資/半退休族群**：北屯十期+8 → CPL 297
3. **科技自營商**：中科水湳中清 → CPL 407~565
4. **金融投資**：大台中 → CPL 645

### 停止投放方向（0 轉換高花費）
- 科技精算師/銀行業 → 0 轉換，此族群單價期望過高
- 3X1 跨興趣測試組 → 大部分 0 轉換，僅「金融」「教育」略有成效
- 韓國流行文化/日漫影視 → 0 轉換，年輕娛樂族群購屋意圖低
- 軍公教/公務員 → 多組 0 轉換，北屯案價對此族群門檻過高

### 下一輪受眾建議
- **科技自營商擴區**：IKEA+15 已有初步成效（CPL 1,069），建議擴大至 COSTCO、好市多範圍
- **中科半退休投資客**：複製「投資/半退休/休閒」模式，擴至中科+15 或洲際+10 範圍
