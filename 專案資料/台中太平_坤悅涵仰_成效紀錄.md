## 策略備註

**建案重點（來自591）**
- 三房 37~41坪，**只有三房**，明確換屋/成長型家庭產品
- 太平國小步行3分鐘，鄰近澄清醫院，家樂福/全聯生活機能完整
- 74號快速道路連接台中市區，但無捷運/高鐵
- 公設比 34~35%，預計 2028 上半年交屋（預售）
- 價格待定，目前階段以收名單為主

**立即行動**
- 換屋/自營商/傳產（5則訊息）最有效 → 建議複製一組調整地區至大里或霧峰，同屬太平生活圈換屋族
- 再行銷 CPL NT$1 極低 → 持續維持，建議補充不同格局/周邊機能素材，強化「太平國小步行3分鐘」學區賣點
- 年輕人興趣組（5潛客）可換素材重測，切角改為「2028完工、現在卡位、74快速道路通台中」

**下一輪受眾方向**
- **學區家長族群**：30-48歲有小孩、關注國小學區的家庭，太平國小步行距離是核心賣點，可搭配「孩子上學走路到」素材
- **大里/霧峰換屋族**：同屬台中外圍區，有換屋需求但不想進市區，訴求「37~41坪三房，同樣預算在太平住得更好」
- **澄清醫院周邊醫護**：離醫院近，訴求「下班5分鐘到家，三房給家人更好的空間」

**停止投放方向**
- 醫生/高管/銀行精算師 → 0 轉換，此案總價/地段不符合這類族群期待，停用
- 親子教育/室內設計/家具傳產 → 0 轉換，興趣標籤與換屋決策行為連結弱，停用
- 74沿線科技主管 → 0 轉換，竹科/科技業族群會優先考慮西屯、北屯，太平方向吸引力不足

**素材建議**
- 主訴求「太平國小步行3分鐘 ＋ 2028完工 ＋ 三房37~41坪」，適合換屋家庭的精準組合
- 價格待定階段：素材不必談錢，改強調「立即預約，搶先選樓層/棟別」製造稀缺感
- 自營商/傳產受眾對「空間換算」敏感，可用「同樣預算，太平比市區多10坪」對比式文案
