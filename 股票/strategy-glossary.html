核心理論
道氏理論
Dow Theory
›
所有技術分析的根基——趨勢的定義、多空轉折的判斷，都源自查爾斯·道的六大原則。
多頭趨勢：每次回檔的低點（L）都比前一個低點高，每次上漲的高點（H）也持續創高
六大核心原則

① 市場反映一切

所有已知和可預見的資訊，已全部反映在股價中。分析師的工作是解讀市場，而非預測。

② 趨勢分三類

主要趨勢（月～年）、次要趨勢（週～月，回檔修正）、短期趨勢（日內～週）。

③ 主要趨勢三階段

累積期（主力吸籌）→ 公眾參與期（散戶進場）→ 分配期（主力出貨）。

④ 指數相互確認

台股：加權指數 + 電子類股指數需同步確認，單一指數突破不算有效突破。

⑤ 成交量確認趨勢

量增價漲 = 趨勢確認；量縮價漲 = 趨勢疑問；量增價跌 = 賣壓沉重。

⑥ 趨勢持續直到明確反轉

不要因為「感覺漲太多了」就逆勢做空。趨勢需要明確的技術訊號才算結束。

實戰判斷法
📌 多頭確認條件（三選三）① 價格創近期新高 ② 回檔低點高於前低 ③ 量增配合漲勢。三個全中 = 安心做多。缺其中一個 = 謹慎。
⚠️ 多頭終結訊號前一個重要低點被有效跌破（收盤在低點之下），代表「高點越來越高、低點越來越高」的格局破壞，多頭趨勢宣告終結。

核心理論
艾略特波浪理論
Elliott Wave Theory
›
市場按「5浪推進＋3浪修正」的節奏循環——理解波浪，你就能預判當前是趨勢浪還是修正浪。
推進浪1、3、5是做多機會；2、4浪是修正不追高；A、C浪是空頭機會或離場時機
三個不可違反的法則
  • 浪2不能跌破浪1的起點（若跌破，重新數浪）
  • 浪3不能是最短的推進浪（1、3、5中）
  • 浪4不能進入浪1的價格區間（重疊，波浪失效）
實戰應用
📌 最重要的機會：浪3浪3是推進浪中最強、最長的浪。識別方式：浪2回檔結束（守住浪1起點之上），MACD金叉配合，量能回升 → 這是波浪理論中最高勝率的買點。
💡 數浪困難怎麼辦波浪理論的最大問題是「事後才看清楚」。實際操作建議：只用波浪確認你已知的趨勢方向，不要只靠波浪猜測未來。週線波浪比日線可靠。

斐波那契回撤
Fibonacci Retracement
›
神奇數字23.6%、38.2%、50%、61.8%是市場最常回撤停留的位置，也是主力最愛設定進出場的關鍵價位。
從低點（100%）到高點（0%）拉斐氏線。回撤到各比例線時觀察是否獲得支撐
五個關鍵比例與意義
比例意義操作
23.6%強勢回檔，趨勢很強可在此加碼，強者恆強
38.2%正常健康回檔多頭第一個觀察支撐
50%心理關卡，半數回吐量縮回踩時買入
61.8%★黃金分割，最重要支撐最高勝率進場點，跌破才認輸
78.6%最後防線跌破代表趨勢可能反轉
正確操作方式
📌 斐氏線怎麼畫上漲趨勢：從最低點拉到最高點。下跌趨勢：從最高點拉到最低點。台股常用「波段低點→波段高點」，不要用日內高低點，要用週線或月線的重要轉折點。
✅ 實戰進場法① 畫出斐氏線 ② 等股價回踩至61.8%區間 ③ 確認K線止跌（錘型、吞噬） ④ 量縮後放量翻紅 ⑤ 進場，停損設61.8%之下2%

技術指標
布林通道
Bollinger Bands
›
中軌是20MA，上下軌是±2倍標準差——通道寬窄反映市場波動率，擠壓後的爆發才是最強訊號。
通道擠壓（變窄）→ 爆量突破一側 → 後續趨勢延伸。這是布林最強的做法。
四種操作訊號

① 布林擠壓後突破（最強）

上下軌收窄到極限，代表波動率降到低點。一旦爆量突破任何一側，是大行情的開始。

② 碰下軌反彈（區間操作）

在震盪格局中，股價碰觸下軌且量縮，隔日出現止跌K線 → 短線買入，目標中軌。

③ 走廊外爆走（強勢股）

股價連續在上軌之上行走 = 超強勢。不要因為「超買」就賣，這是主力鎖倉拉高。

④ W底+中軌確認

布林下軌出現W底，第二個底守住且股價突破中軌（20MA）→ 是轉強的確認訊號。

公式
中軌 = 20日收盤價均值（20MA） 上軌 = 中軌 + 2 × 20日收盤價標準差 下軌 = 中軌 − 2 × 20日收盤價標準差 理論上：95%的價格在上下軌之間 帶寬 = (上軌 - 下軌) / 中軌（帶寬越小=擠壓越緊）

技術指標
維加斯通道
Vegas Tunnel
›
用144和169兩條EMA形成「隧道」，股價在隧道上方做多、下方做空，回踩隧道時進場——簡單卻有效。
144 EMA（藍）和 169 EMA（紫）形成隧道。股價在隧道上方每次回踩都是加碼機會。
操作規則
  • 144 EMA 在 169 EMA 之上 = 多頭格局，只做多
  • 股價在隧道之上 + 回踩至隧道附近 = 最佳買點
  • 進場：K線止跌（錘型/吞噬）+ 量縮後放量
  • 股價跌入隧道內部 = 多空不明，等待方向確認
  • 股價在隧道之下 = 做空格局，不做多
💡 為何是144和169144 = 12×12（一年交易週數×月數），169 = 13×13。這兩個數字都是斐波那契數列的倍數，在外匯和商品期貨市場有大量機構參與者使用，形成自我實現的支撐/壓力。台股因外資影響，也有一定效果，但效果不如外匯市場穩定。

型態辨識常見難點
W底 ＆ M頭完整判斷法
Double Bottom & Double Top
›
看不出W底和M頭，大多是因為進場時機搞錯了——真正的進場點不在底部，而在突破頸線的那一刻。
W底進場 = 收盤突破頸線；M頭放空 = 收盤跌破頸線。兩者都不在頂底進場。
W底逐步辨識清單（按順序確認）
  • 股價處於明顯的下跌趨勢後（不是橫盤出現的W）
  • 出現第一個底部，通常伴隨較大的恐慌性成交量
  • 股價從第一個底反彈 10%~30%，形成「頸線」（兩底間最高點）
  • 第二個底部的低點 ≤ 第一個底（不比第一個低太多）
  • 第二個底的成交量明顯小於第一個底（最重要！）
  • 等股價收盤突破頸線（不是盤中短暫穿過）才進場
  • 突破頸線當天量能放大 = 高確信突破
W底 vs 假W底：如何區分
真W底 ✅假W底 ❌
第二底位置≤ 第一底（不創新低）明顯低於第一底
第二底量能萎縮（<第一底量）放大或相近
反彈形態頸線附近有整理一路急漲急跌
頸線突破量增收盤在頸線上盤中穿過隨即跌回
操作突破頸線買入繼續觀望
目標價計算與停損設定
W底目標價 = 頸線價格 + (頸線 − W底部) 範例：頸線 = 100元，W底 = 85元 目標 = 100 + (100 − 85) = 115元 停損設定：跌破頸線就出場 （確認是假突破，立刻停損） M頭目標價 = 頸線價格 − (M頂部 − 頸線) 範例：頸線 = 100元，M頂 = 115元 目標 = 100 − (115 − 100) = 85元
最常見的三個錯誤
❌ 錯誤1：在第二個底就買入這是最常見的錯誤。你以為找到底部，但股價可能繼續創低。只有突破頸線才代表趨勢轉折確認。
❌ 錯誤2：只看形狀不看量W底必須配合量縮的第二個底。如果第二個底量比第一個底還大，代表賣壓仍在，W底不成立。
❌ 錯誤3：用盤中高點當「突破」突破頸線必須是收盤價在頸線之上，盤中短暫穿越後又跌回去不算突破，這是假突破陷阱。
M頭判斷（W底鏡像邏輯）
  • 股價處於明顯的上漲趨勢後（不是橫盤中出現）
  • 第一個頂部伴隨量大，回檔後形成頸線
  • 第二個頂的高點 ≥ 第一個頂（不突破新高）
  • 第二個頂的量明顯縮小 = 多頭力量衰竭
  • 收盤跌破頸線 + 量放大 = 放空或離場訊號
  • 回測頸線（跌破後反彈至頸線附近）= 最佳放空點

型態辨識常見難點
缺口理論完整解析
Gap Theory — 4 Types
›
缺口就是兩根K線之間完全沒有交易的空白區域。看到缺口別慌，先判斷是哪一種，才能決定要追還是要跑。
從左到右：普通缺口（無意義）、突破缺口（做多）、逃逸缺口（繼續持有）、竭盡缺口（減碼）
四種缺口一覽表
類型出現位置成交量是否回補交易意義
普通缺口盤整區間中正常✅ 快速回補無意義，不操作
突破缺口盤整後突破時爆量❌ 不回補⬆️ 趨勢啟動，追入
逃逸缺口趨勢中段放量❌ 不回補⬆️ 繼續持有/加碼
竭盡缺口趨勢末段放量後萎縮✅ 快速回補⚠️ 準備離場
逃逸缺口測量目標價
📌 逃逸缺口又叫「量測缺口」因為它通常出現在整段趨勢的中點。公式：
目標價 = 逃逸缺口位置 + (逃逸缺口位置 − 起漲點)
例：起漲 100元 → 逃逸缺口在 130元 → 目標 130 + (130−100) = 160元
竭盡缺口的辨識：最難但最重要
  • 股價已大幅上漲一段（通常>30%）
  • 出現跳空向上的缺口
  • 跳空當天量極大（全天最大量之一）
  • 當天K線收盤接近開盤價（盤中拉高後賣壓出現）
  • 隔天或兩三天內股價跌回缺口 = 缺口開始被填補
  • 一旦確認是竭盡缺口：減碼出場，不要等
⚠️ 除權除息的假缺口股票除息日當天會出現向下跳空，這不是真正的技術缺口，而是股價調整配息金額的結果。分析時必須使用「還原除息報價」，才不會誤判缺口訊號。
判斷突破缺口 vs 竭盡缺口
💡 三天觀察法跳空後觀察三天：① 若股價持續在缺口之上，且量能維持 = 突破缺口，繼續做多。② 若股價在1~3天內跌回填補缺口 = 竭盡缺口，立刻出場。③ 不確定時：先減半部位，等方向明確再決定。

型態辨識常見難點
旗形與三角旗型態
Flag & Pennant Pattern
›
旗形是最穩定的連續型態，等旗面整理結束後突破，趨勢繼續延伸。但大多數人等突破才買，其實有更好的進場點。
旗形最佳進場是「回踩旗頂線+量縮」，比等突破點更早且風險更低
旗形辨識完整清單
  • 有一段明顯的快速急漲（旗桿），通常3~10天，漲幅>15%
  • 急漲後出現2~4週的小幅回調整理（旗面），量明顯萎縮
  • 旗面的兩條邊線平行（旗形）或收斂（三角旗）
  • 旗面最大回調幅度不超過旗桿的50%（若超過代表型態破壞）
  • 整理期間量能比旗桿時明顯縮小（必要條件）
  • 若量在旗面期間放大，代表有賣壓，型態可能失效
兩種進場方式比較
⭐ 最佳進場：回踩旗頂線一般進場：突破旗面
時機旗面整理期間，股價跌回旗桿頂點位置股價放量突破旗面上緣
停損設旗面底部（旗桿頂點下方2%）設突破點下方2%
風險報酬更好（買在更低處，停損更近）較差（追高進場）
難度需要耐心等待容易執行但常追高
確認條件回踩後量縮，K線止跌（錘型）突破時量需放大
目標價與停損計算
旗桿長度 = 旗桿頂部 − 旗桿起點 目標價 = 突破點 + 旗桿長度 範例： 旗桿起點 = 80元，旗桿頂部 = 100元 旗桿長度 = 100 − 80 = 20元 突破點 = 98元（突破旗面上緣） 目標價 = 98 + 20 = 118元 最佳進場停損： 進場 = 回踩旗頂線（約100元） 停損 = 旗面底部（例如93元），即-7% 目標 = 118元，獲利+18% 風險報酬比 ≈ 1:2.5（非常理想）
旗形失效的訊號
⚠️ 旗面回調超過旗桿50%旗形整理期間，如果股價跌回旗桿漲幅的50%以下，代表賣壓過強，買方無力守住，此旗形很可能失效。此時需觀望，不要強行進場。
⚠️ 旗面量持續放大正常旗形整理時量會萎縮。如果旗面期間量越來越大，代表有人在高位出貨，這不是整理，而是出貨型態。

基礎工具
支撐與阻力
Support & Resistance
›
支撐就是「買方牆」，阻力就是「賣方牆」——學會找關鍵位置，進場點和停損點自然清晰。
五種重要支撐/阻力來源

① 前高/前低

曾經的高點會成為壓力，曾經的低點會成為支撐。「角色互換」：突破前高後，前高變支撐。

② 整數關卡

100、200、500、1000元等整數，因為心理因素會有大量掛單。台積電的整百元是強支撐壓力。

③ 均線

20MA、60MA、120MA是主力的平均成本線，多頭格局中是支撐，空頭格局中是壓力。

④ 成交量密集區

過去大量成交的價格區間，有大量套牢或獲利的人，會形成強力支撐（解套賣）或壓力（獲利了結）。

💡 支撐壓力角色互換定律被有效突破的阻力 = 新的支撐；被有效跌破的支撐 = 新的阻力。這是技術分析最核心的一個規律，適用於所有型態。頸線突破後回測就是這個道理。

基礎工具
趨勢線畫法
Trend Line
›
趨勢線需要至少兩個接觸點才有效；接觸點越多、維持時間越長，突破的意義越大。
畫趨勢線的規則
  • 上升趨勢線：連接兩個以上的「波段低點」
  • 下降趨勢線：連接兩個以上的「波段高點」
  • 不要強迫連線——若連線需要穿過K線實體，這條趨勢線無效
  • 有3個以上的接觸點 = 強力趨勢線，突破更有意義
📌 趨勢線突破的確認收盤突破趨勢線（不是盤中），且突破時量能放大。突破後回測趨勢線（原來的支撐變壓力，或原壓力變支撐）= 確認進場點。
