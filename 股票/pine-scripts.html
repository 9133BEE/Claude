//@version=5 indicator("跳空壓力支撐區間", overlay=true, max_boxes_count=200) min_gap_pct = input.float(0.1, "最小缺口比例(%)", minval=0.01, step=0.05) / 100 max_bars = input.int(100, "最多保留幾根K棒", minval=10) col_up_gap = input.color(color.new(color.green, 80), "跳空支撐填色") col_dn_gap = input.color(color.new(color.red, 80), "跳空壓力填色") border_up = input.color(color.green, "跳空支撐邊框") border_dn = input.color(color.red, "跳空壓力邊框") // 跳空向上（支撐缺口）: 本K低點 > 前K高點 gap_up = low > high[1] and (low - high[1]) / high[1] >= min_gap_pct // 跳空向下（壓力缺口）: 本K高點 < 前K低點 gap_dn = high < low[1] and (low[1] - high) / low[1] >= min_gap_pct if gap_up box.new(bar_index - 1, low, bar_index + max_bars, high[1], bgcolor=col_up_gap, border_color=border_up, border_width=1) if gap_dn box.new(bar_index - 1, low[1], bar_index + max_bars, high, bgcolor=col_dn_gap, border_color=border_dn, border_width=1) // 缺口填補後延伸框至當前 bar var boxes = array.new_box() array.push(boxes, gap_up ? box.new(bar_index,low,bar_index,high[1],border_color=color.na) : box.na) plotshape(gap_up, style=shape.triangleup, location=location.belowbar, color=color.green, size=size.tiny, title="跳空支撐") plotshape(gap_dn, style=shape.triangledown, location=location.abovebar, color=color.red, size=size.tiny, title="跳空壓力")

使用說明

  • 綠框 = 跳空支撐（回測此區域時考慮買進）
  • 紅框 = 跳空壓力（上漲遇此區域考慮減碼）
  • 調整「最小缺口比例」可過濾雜訊（建議 0.1–0.3%）
  • 「最多保留幾根K棒」決定缺口框延伸長度

使用說明

  • 貼到 TradingView → Pine 編輯器 → 儲存 → 加到圖表
  • Auto 模式：切換圖表時框時自動調整樞紐與搜尋參數，無需手動設定
  • 進場邏輯：等待 MSS（結構突破）→ 找到 OB → 價格回測 OB 區間 → ▲/▼ unicorn 出現再進場
  • 出場邏輯：結構反轉時自動出現「⚠ 多單/空單出場」標籤，提示平倉時機
  • +OB / -OB x/4：品質分數，3分以上升級為 HQ（紫色），優先交易高分 OB
  • HTF OB 方框（1H/4H/1D）：大時框訂單塊，可作為目標區間或進場參考
  • HTF FVG 方框（虛線）：大時框公平價值缺口，被填滿後自動消失
  • 止損建議放在 OB 另一端（頂部或底部）外側 1–2 個 tick
